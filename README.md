Contract Form Website

A simple **Contract Form** project built using **HTML, CSS, and JavaScript**.  
This project contains a clean user interface with input fields for:

- Name
- Email
- Phone Number
- Submit Button

---

## 📌 Features

- Simple and responsive design
- Clean UI layout
- Basic HTML structure
- CSS styling
- JavaScript functionality
- Beginner-friendly project

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
